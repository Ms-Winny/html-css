Criando Formulários

Para criar um formulário basta usar as tags form e /form

Os formulários precisam ter um id e um nome para serem identificados.

Além disso, é essencial envelopar as perguntas do formulário com a tag label para conectar uma pergunta com a caixa de input.

Passo a passo

Primeiramente, ao criar um formulário com a tag form o visual studio irá criar um form com um único parâmetro, o action="". Nós iremos o ignorar por enquanto, pois é necessário o conhecimento sobre outra linguagem.
