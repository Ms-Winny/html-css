Criando Formulários

Para criar um formulário basta usar as tags form e /form

Os formulários precisam ter um id e um nome para serem identificados.

Além disso, é essencial envelopar as perguntas do formulário com a tag label para conectar uma pergunta com a caixa de input.

Passo a passo

Primeiramente, ao criar um formulário com a tag form o visual studio irá criar um form com um único parâmetro, o action="". Nós iremos o ignorar por enquanto, pois é necessário o conhecimento sobre outra linguagem.

Atributos de Formulários

Usuários, Senhas e Campos Obrigatórios

Existem formulários aonde os dados pessoais são sensíveis e não podem estar visíveis, para isso usaremos o input type="password" para que os dados fiquem visualmente ocultos.

Nos formulários que criamos até agora, é possível que o usuário envie o o formulário sem preencher os dados, o que é um problema para dados obrigatórios. Para resolver isso colocamos a palavra required no input obrigatório.



Com o atributo size="" definimos o tanto de caracteres que a caixa mostra sem precisar de barra de rolagem. Sabe aquele texto em letras transparentes que servem para guiar o usuário a saber o que deve digitar em cada caixa? Pois é, também é possível colocá-los, basta o usar o placeholder="".

Além do mais, também há a possibilidade de atribuir limites minímos e máximos de caracteres, bastar usar minlenght="" e maxlenght="".
