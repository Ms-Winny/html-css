Estudo de Tabelas

Para criar uma tabela é necessário conhecimento de algumas tags:

table = Tabela

tr = Table Row ou Linha de Tabela

th = Table Header ou Cabeçalho de Tabela

td = Table Data ou Dado de Tabela

Com essas informações, vamos criar uma tabela com alguns dados fictícios:

Primeiro utilizarei a tag 'table' e dentro dela as tags 'tr' para criar as linhas
