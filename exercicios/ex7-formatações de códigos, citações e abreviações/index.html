Código-fonte

O comando document.getelementById('teste') é escrito em linguagem javaScript.

Acima usei uma fonte mono-espaçada para digitar o código, essa fonte é excelente para ler códigos. Aqui vai mais um exemplo:

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'0 número {num} é PAR')
    else:
        print(f'0 número {num} é ÍMPAR')
    print('Fim do programa')

Para formatar usei a tag <code></code>.
